# EVE 藏宝阁检索系统 V2.2 更新说明

### 新功能

**1. 纳米核心关键字筛选**

新增「纳米核心」过滤条件，支持通配符 `*` 匹配：
- `元帅级*` — 匹配所有以"元帅级"开头的纳米核心
- `*碎星*` — 匹配所有包含"碎星"的纳米核心
- `*级*核心` — 匹配所有拥有纳米核心的账号
- `*级*智能核心` — 匹配所有拥有智能纳米核心的账号

在筛选面板点击 **「纳米核心」** 按钮即可配置关键字规则，多条规则为 AND 关系（需同时满足）。匹配内容在详情视图中以**亮绿色**高亮显示。

**2. 高亮导航**

详情视图新增导航按钮：
- **「◀ 上一个」** / **「下一个 ▶」** — 在筛选关键字的所有高亮位置之间快速跳转
- 导航标签实时显示当前高亮位置（如「元帅级碎星核心 2/5」）和总匹配数

选择角色后不再需要手动滚动查找匹配内容，一键即可定位。

**3. 版本自动更新检查**

程序启动时自动检测 GitHub 上的最新版本：
- 通过 `git ls-remote --tags` 查询远程标签
- 备选方案：通过 GitHub Releases API 获取最新发布版本
- 检测到新版本时弹出提示，可选择直接跳转到 GitHub 页面下载最新版本

**4. 筛选条件颜色标识**

各筛选条件在详情视图中的高亮颜色现已区分：
- **黄色** — 技能关键字 / 全局关键字匹配
- **亮绿色** — 纳米核心关键字匹配
- **淡蓝色** — 植入体规则匹配

筛选面板的按钮标签也已标注对应颜色，便于对照。
