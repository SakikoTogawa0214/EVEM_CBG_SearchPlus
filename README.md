# EVE 藏宝阁本地检索工具

这是一个为 EVE Echoes（星战前夜·无烬星河）玩家设计的藏宝阁角色检索工具。它通过爬取藏宝阁网站上的角色数据，并提供一个图形用户界面（GUI），让用户可以根据多种条件在本地快速检索和筛选角色，从而找到心仪的账号。

## 主要功能

- **数据爬取**：
  - 自动调用 Chrome 浏览器，模拟用户操作，从藏宝阁网站爬取角色详细数据。
  - 支持设置爬取次数（或无限循环爬取），增量更新本地数据库。
  - 数据以 JSON 格式保存在本地（`eve_full_data.json`），方便后续处理。

- **多维度检索**：
  - **基础数值筛选**：按最高价格、总技能点（万）、最低保险点进行筛选。
  - **进阶点数筛选**：按最低洗点点数（万）、最低自由技能点（万）进行筛选。
  - **关键字筛选**：
    - **技能关键字**：筛选必须包含某些特定技能的角色。
    - **全局关键字**：在角色的所有数据中进行模糊搜索。
  - **植入体规则**：强大的植入体筛选功能，可以自定义规则，例如：
    - 要求必须包含特定名称的植入体。
    - 要求植入体的型号（如“标准型”、“进阶型”）。
    - 要求植入体的最低等级。
    - 支持设置多个植入体规则，角色必须同时满足所有规则。
  - **纳米核心过滤条件**：支持通配符 `*` 匹配

- **图形用户界面 (GUI)**：
  - 使用 Tkinter 构建，界面直观，易于上手。
  - 实时显示检索结果列表，包含角色编号、昵称、价格、技能点等关键信息。
  - 选中角色后，可在右侧详细视图中查看完整的 JSON 数据。
  - 关键字和植入体在详细视图中会高亮显示，方便快速定位。
  - 双击列表中的角色，可直接在浏览器中打开对应的藏宝阁页面。
  - 提供实时运行日志，方便跟踪爬取状态和调试。

- **环境配置**：
  - 提供“检测环境”功能，一键检查 Chrome 浏览器是否已开启调试模式。
  - 如果环境未就绪，可引导用户自动启动配置好参数的 Chrome 浏览器实例。

## 如何使用

1. **环境准备**：
   - 下载（ [Evem_CBG_SearchPlus_V2.2.exe](https://github.com/SakikoTogawa0214/EVEM_CBG_SearchPlus/blob/main/dist/EVEM_CBG_SearchPlus_V2.2.exe) ）
   - 确保已安装 Google Chrome 浏览器。
   - 运行 `EVE藏宝阁检索系统V2.2` 启动程序。

3. **启动浏览器并登录**：
   - 点击 **[🌐 检测环境]** 按钮。如果提示“未连接”，程序会询问是否启动 Chrome。
   - 在新打开的 Chrome 窗口中，手动登录你的藏宝阁账号。这是为了让爬虫能够访问到需要登录才能查看的数据。

4. **开始爬取数据**：
   - 登录成功后，回到工具界面，点击 **[⚡ 开始爬取]** 按钮。
   - 程序会提示输入拉取次数（0 代表无限次），根据需要输入后即可开始自动爬取。
   - 爬取过程会在下方的“实时运行状态”窗口显示日志。

5. **进行检索**：
   - 在“筛选与控制面板”中设置你的筛选条件。
     - **数值类**：直接在输入框填入数字。
     - **关键字/植入体**：点击对应的配置按钮，在弹出的窗口中添加、修改或删除规则。
   - 设置完毕后，点击 **[🔍 立即检索]** 按钮，符合条件的角色会立即显示在下方的列表中。
   - 点击 **[🔄 重置]** 可以清空所有筛选条件。

6. **查看与跳转**：
   - 单击列表中的任意角色，右侧会显示该角色的所有详细数据。
   - 双击列表中的任意角色，会自动用默认浏览器打开该角色的藏宝阁购买页面。

## 技术栈

- **Python 3**
- **Tkinter**：用于构建图形用户界面。
- **Playwright / Requests**：用于与 Chrome 浏览器交互和执行网络爬取。
- **JSON**：用于数据存储。
